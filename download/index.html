This Chrome-Addon still isn't in the Google Web-Store. Therefore the installing process is not as straight forward as you are used to. I hope this install-guide will make it still easy for you to use this Addon!

How to install the Addon?

  1. Download the ZIP-File
  2. Create a folder where you would like to save the extension
  3. Unpack the ZIP-File there
  4. Head in Chrome to this URL: chrome://extensions/
  5. Enable Developer mode in the top-right corner of that screen
  6. Click on "LOAD UNPACKED" and click on the folder you chose to store the Addon
  7. If it's not enabled immediately, enable it.

In the Top-Right Corner of your browser will be a Live-Lyrics Icon. Once you click on that, a small window will open. In that window you can see the currently recogniced Song, you can enable/disable the Plugin and you can choose to support me. More information here.

FAQ

  • Do you safe any data?
  • No, I do not. Spotify is saving the number of requests, but thats it.
  • Why do I have to log-in every time?
  • That's because I don't want to save your log-in data. The Spotify authorization has a time limit, so there currently is no way arount it.
